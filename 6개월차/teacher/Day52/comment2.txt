데이터 삽입 : INSERT 문 

INSERT 테이블명 (컬럼들)
	VALUES (컬럼값)
	
여러줄 INSERT 

INSERT 테이블명 (컬럼들)
	VALUES 	(컬럼값),
			(컬럼값)
	
	

데이터 조회 : SELECT 문

SELECT [컬럼명], ....  혹은 *
	FROM 테이블명
	

조회를할때, [필터, 정렬]

필터기능 : WHERE 
 사용하는 부등호 : <, >, <=, >= , =, <>, BETWEEN AND, IN 
 
 
정렬기능 : ORDER BY, ORDER BY DESC


테이블끼리 조인 : JOIN 문




조회는 더이상 배울게없습니다.

UPDATE 문
DELETE 문 

UPDATE 문 : 컬럼의 값을 수정하는 명령어
UPDATE 테이블명
	SET 컬럼명 = 값
	WHERE 조건
	
	
DELETE 문 : 컬럼의 ROW를 삭제하는 명령어

DELETE FROM 테이블명 
	WHERE 조건
